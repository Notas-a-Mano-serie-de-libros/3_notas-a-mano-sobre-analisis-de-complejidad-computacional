<h2>🧠 Estructura conceptual del libro</h2>

<ul>
  <li><p style="text-align:justify;"><b>Modelos de cómputo:</b> fundamentos de la máquina de Turing, RAM y modelos alternativos de ejecución.</p></li>
  <li><p style="text-align:justify;"><b>Medición del costo computacional:</b> análisis del tiempo y espacio de ejecución, funciones de complejidad y modelos de costo unitario.</p></li>
  <li><p style="text-align:justify;"><b>Notaciones asintóticas:</b> uso formal de 𝒪, Ω y Θ para describir el comportamiento de los algoritmos en el límite.</p></li>
  <li><p style="text-align:justify;"><b>Análisis amortizado:</b> estudio del costo promedio de una secuencia de operaciones; métodos del agregado, contable y potencial.</p></li>
  <li><p style="text-align:justify;"><b>Clasificación de problemas:</b> clases de complejidad P, NP, NP-completo y NP-difícil; noción de reducibilidad polinómica.</p></li>
  <li><p style="text-align:justify;"><b>Complejidad espacio-temporal:</b> relaciones entre el tiempo de ejecución y los recursos requeridos.</p></li>
  <li><p style="text-align:justify;"><b>Estructuras de datos eficientes:</b> implementación y análisis de listas, pilas, colas, árboles, montículos, tablas hash y grafos.</p></li>
  <li><p style="text-align:justify;"><b>Teoría de la computabilidad:</b> problemas decidibles e indecidibles; funciones recursivas y lenguajes reconocibles.</p></li>
</ul>

<h2>⚙️ Requisitos previos</h2>

<ol>
  <li><b>Python 3.8 o superior</b> (recomendado 3.10+)</li>
  <li><b>Entorno Jupyter</b> (JupyterLab o soporte de notebooks en el IDE)</li>
  <li><b>IDE recomendado:</b> VSCode o IntelliJ IDEA (con plugins de Python y Jupyter)</li>
</ol>

<p style="text-align:justify;">
Los ejemplos del libro se distribuyen como notebooks interactivos <code>.ipynb</code> que pueden ejecutarse localmente o en entornos en la nube como <b>Google Colab</b>.  
Para un entorno local, se recomienda instalar las librerías básicas de ciencia de datos:
</p>

```bash
pip install jupyterlab voila ipywidgets numpy pandas matplotlib networkx
```

<p><b>Verificar versiones:</b></p>
```javapython --version
pip --version
jupyter lab --version
```
